We used different conditions and saw the accuracy for each of these models:

(A) Using LinearRegression
1) Very basic LinearRegression model with no date and removing all the rows with missing data: 65.00%
2) Added date to the above model but took only the month number: 65.12%
3) Added day as a fraction to the month to the above model, like January 2nd is 1 + 1/31: 65.13%
4) Use complete data as long as we have the AQI (output) and replaced missing cells with ffill and bfill: 58.47%
5) Use above model but use rows with not more than 3 columns missing: 60.58%

(B) Using SVM
6) Use the earlier model only with rows with complete information: 81.20%
7) Use the model with rows having not more than 3 columns missing: 75.30%

(C) Using KNN
8) Using KNN on complete row dataset: 84.71%
9) Using KNN on incomplete-now-filled dataset: 78.37%

(D) Using Deep Learning
10) Using Deep Learning on complete row dataset, two hidden layers: ~85%
11) Using Deep Learning on incomplete-now-filled dataset, two hidden layers: ~70%
